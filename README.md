# 概要
エクセルアドインです。  
ExcelからCSVに変換します。
- オーダーシートExcelから投入表インポートCSV形式へ変換

# インストール
1. インストーラを[こちら](https://github.com/GangDuo/excel-addin-new-item-po-csv-gen/releases)からダウンロードします。
1. ダウンロードしたファイルを実行します。
1. 指示に従いインストールします。
